# قائمة إطلاق مشروع دار شمس التعافي (Launch Checklist)

هذه القائمة يجب مراجعتها بالكامل قبل تسليم النظام للعمل الفعلي.

## ✅ البنية التحتية (Infrastructure)
- [ ] مشروع Supabase نشط ومربوط.
- [ ] تم رفع `schema.sql` بنجاح.
- [ ] تم إنشاء حساب Super Admin أول وتغيير كلمة مروره.
- [ ] تم إعداد Storage Buckets (`gallery` و `residents`).

## ✅ لوحة التحكم (Dashboard)
- [ ] تم رفع الـ Environment Variables (`URL`, `ANON_KEY`).
- [ ] الـ Middleware مفعل ويمنع الدخول بدون صلاحية.
- [ ] تم اختبار إضافة مقيم وتعديله.
- [ ] تم اختبار نشر خبر جديد وظهوره في الموبايل.

## ✅ تطبيق الموبايل (Flutter App)
- [ ] تم تغيير اسم التطبيق إلى "دار شمس".
- [ ] تم تغيير أيقونة التطبيق بشعار المصحة.
- [ ] تم تغيير الـ Package Name ليكون فريداً (مثلاً: `com.darshams.app`).
- [ ] تم إنشاء نسخة الـ APK و AppBundle النهائية.

## ✅ الخصوصية والقانون (Legal & Privacy)
- [ ] إضافة صفحة "سياسة الخصوصية" داخل التطبيق (خاصة ببيانات المرضى).
- [ ] التأكد من عدم تخزين أي كلمات مرور في قاعدة البيانات بشكل نصي (Supabase Auth يتكفل بذلك).
- [ ] مراجعة شروط الاستخدام للأهالي.

## ✅ خطة الدعم والطوارئ
- [ ] وجود نسخة احتياطية (Backup) من قاعدة البيانات.
- [ ] رقم الدعم الفني (01115540077) يعمل ومربوط بأزرار الاتصال.
- [ ] بريد الدعم الفني مفعل لاستقبال الرسائل من لوحة التحكم.

## 🚀 خطوات النشر النهائية
1. ارفع الـ Dashboard على **Vercel**.
2. ارفع الـ APK على **Google Play Console**.
3. ارفع نسخة الـ iOS على **App Store Connect**.
4. قم بإرسال بيانات الدخول لأول موظف لبدء إدخال البيانات الحقيقية.
